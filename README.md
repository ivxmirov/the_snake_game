## Быстрый старт

1. Клонируйте репозиторий

```bash
git clone https://github.com/ivxmirov/the_snake_game.git
```

2. Установите зависимости (включая dev-зависимости)

```bash
uv sync --group dev
```

3. Активируйте виртуальное окружение

```bash
source .venv/Scripts/activate
```

4. Запустите файл the_snake.py
  
```bash
python the_snake.py
```

5. Наслаждайтесь игрой !
